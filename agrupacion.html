Las etiquetas de agrupación

Las etiquetas de agrupación en HTML permiten organizar y estructurar el contenido de una página web de manera lógica.
Estas etiquetas son útiles para crear secciones, dividir contenido y mejorar la accesibilidad y la presentación A continuación,
te explico las principales etiquetas de agrupación:

Principales Etiquetas de Agrupación en HTML

  1. LA Etiqueta DIV
    • La etiqueta < div > es un contenedor genérico que agrupa contenido. Se utiliza combinado con CSS para aplicar
      estilos a bloques de contenido.
    • No tiene un significado semántico por sí mismo, pero permite agrupar elementos para facilitar el diseño.
  2. LA Etiqueta span
    • La etiqueta < span > es un contenedor en línea que permite agrupar texto o elementos sin alterar su flujo. Es útil
      para aplicar estilos específicos a partes del texto.
  3. LA Etiqueta < section >
    • La etiqueta < section >representa una sección genérica de contenido en un documento. Se utiliza para agrupar
      contenido temático y generalmente debe incluir un encabezado.
  4. LA Etiqueta < article >
    • La etiqueta < article >representa un contenido independiente que podría ser distribuido o reutilizado de forma
      independiente, como un artículo de blog, un comentario o una noticia.
  5. LA Etiqueta < footer >
    • La etiqueta < footer >representa un bloque de contenido que se ubica al final de la página y se utiliza para
      proporcionar información sobre el contenido de la página, como un enlace a un sitio web o un enlace a un archivo PDF.
  6. LA Etiqueta < header >
      7. La etiqueta < header > se utiliza para definir el encabezado de un documento o una sección, incluyendo logotipos,
      títulos y enlaces de navegación.

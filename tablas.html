Las Tablas

Las tablas se utilizan para organizar datos en filas y columnas, ideales para mostrar información estructurada como horarios,
listas de precios o informes. Aunque no son el método preferido para crear el diseño general una página, siguen siendo útiles
para la presentación de datos tabulares.

    Ejemplo de tabla

  • Estructura Básica de una Tabla HTML
    1. La etiqueta <table> define una tabla.
    2. <tr> (fila de la tabla): Define una fila en la tabla.
    3. <th> (encabezado de tabla): Defina una celda de encabezado en una fila.
    4. <td> (datos de tabla): Defina una celda de datos en una fila.

    Atributos y Propiedades Útiles

    • border: Defina el grosor del borde de la tabla (aunque es mejor usar CSS para esto).
    • colspan: Permite que una celda abarque varias columnas.
    • rowspan: Permite que una celda abarque varias filas.
